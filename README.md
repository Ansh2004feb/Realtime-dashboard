# 📊 Real-Time System Integration Dashboard

A real-time monitoring dashboard built with Node.js and Socket.IO that demonstrates **system integration capabilities** through live data streaming, dynamic visualizations, and automated alerting.

## ✨ Features

- **Live Data Streaming**: Generates simulated system metrics every 3 seconds
- **Real-time Updates**: WebSocket connection pushes data instantly to all clients
- **Interactive Charts**: Live line and bar charts using Chart.js
- **Smart Alerting**: Color-coded status indicators (🔴 HIGH / 🟡 MEDIUM / 🟢 LOW)
- **System Monitoring**: Uptime counter, peak value tracking, and recent alerts log
- **Responsive Design**: Works on desktop and mobile devices

## 🛠️ Tech Stack

- **Backend**: Node.js, Express.js, Socket.IO
- **Frontend**: HTML5, CSS3, JavaScript
- **Visualization**: Chart.js
- **Real-time Communication**: WebSockets

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/anshgoel/realtime-dashboard

# Install dependencies
npm install

# Start server
npm start

# Open in browser
http://localhost:3000
```
